Wwise SDK 2023.1.0 - Windows
Creating new Source Plug-ins for a Voice

Creates two new Source objects for a Voice using plug-in Silence and SoundSeed Grain for the languages "English(US)" and "French".

Function URI

ak.wwise.core.object.set

Arguments

{
"objects": [
{
"object": "\\Actor-Mixer Hierarchy\\Default Work Unit\\MyVoice",
"children": [
{
"type": "Source",
"name": "source1",
"language": "English(US)",
"classId": 6619138
},
{
"type": "Source",
"name": "source2",
"language": "French",
"classId": 11993090
}
]
}
]
}

Options

{
"return": [
"id",
"name"
]
}

Result

{
"objects": [
{
"id": "{7ACD763F-2E44-4D07-AA7F-E6C9C60B2DDC}",
"name": "VO",
"children": [
{

},
{

}
]
}
]
}
