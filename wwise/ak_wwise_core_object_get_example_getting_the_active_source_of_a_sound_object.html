Wwise SDK 2023.1.0 - Windows
Getting the active source of a sound object

Uses the "activeSource" accessor to get the sound's active source.

Function URI

ak.wwise.core.object.get

Arguments

{
"waql": "\"{24979032-B170-43E3-A2E4-469E0193E2C4}\""
}

Options

{
"return": [
"id",
"name",
"activeSource"
]
}

Result

{
"return": [
{
"id": "{24979032-B170-43E3-A2E4-469E0193E2C4}",
"name": "My Tone",
"activeSource": {
"id": "{B7026191-28B7-4CFF-AE5A-4F0FD63607EC}",
"name": "sine_880hz_48k_21ms"
}
}
]
}

Refer to Using the Wwise Authoring API (WAAPI) for more information about Wwise Authoring API.
