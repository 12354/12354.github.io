Wwise SDK 2023.1.0 - Windows
Pasting a property and all lists

Paste the property, reference and list differences from the source object to the target objects. Only the "Volume" property will be pasted along with all lists for the source object. The source object only has "RTPC" and "Metadata" lists.

Function URI

ak.wwise.core.object.pasteProperties

Arguments

{
"source": "\\Actor-Mixer Hierarchy\\Default Work Unit\\Source Sound",
"targets": [
"\\Actor-Mixer Hierarchy\\Default Work Unit\\Target Sound",
"\\Actor-Mixer Hierarchy\\Default Work Unit\\Target Sound2"
],
"inclusion": [
"Volume",
"RTPC",
"Metadata"
]
}

Result

{}

Refer to Using the Wwise Authoring API (WAAPI) for more information about Wwise Authoring API.
