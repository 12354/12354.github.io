Wwise SDK 2023.1.0 - Windows
ak.wwise.core.sourceControl.delete

Delete files from source control. Equivalent to Mark for Delete for Perforce.

Arguments

Name Type Description
files * array Array of files to delete. File paths must be absolute.
Several values provided for the option.
files [...] string A value for the option.

(Required *)
