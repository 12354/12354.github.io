Wwise SDK 2023.1.0 - Windows
Importing a WAV file into a new Sound SFX and creating the associated Event

Creates a new Sound object, imports a WAV file into it, and then creates an Event with a Play action on the Sound.

Function URI

ak.wwise.core.object.set

Arguments

{
"objects": [
{
"object": "\\Actor-Mixer Hierarchy\\Import",
"children": [
{
"type": "Sound",
"name": "MySound",
"import": {
"files": [
{
"audioFile": "C:\\MySound.wav",
"originalsSubFolder": "SUB"
}
]
}
}
]
},
{
"object": "\\Events\\Default Work Unit",
"onNameConflict": "merge",
"children": [
{
"type": "Event",
"name": "Play_MySound",
"children": [
{
"name": "",
"type": "Action",
"@ActionType": 1,
"@Target": "\\Actor-Mixer Hierarchy\\Import\\MySound"
}
]
}
]
}
],
"onNameConflict": "merge",
"listMode": "replaceAll"
}

Options

{}
