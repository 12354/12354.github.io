◆ RegisterGameObj() [2/2]

AKSOUNDENGINE_API AKRESULT AK::SoundEngine::RegisterGameObj ( AkGameObjectID  in_gameObjectID,
const char *  in_pszObjName 
)

Registers a game object.

Returns
  • AK_Success if successful
  • AK_InvalidParameter if the specified AkGameObjectID is invalid. Range 0xFFFFFFFFFFFFFFE0 (-32) to 0xFFFFFFFFFFFFFFFF (-1) are invalid inclusively.
Remarks
Registering a game object twice does nothing. Unregistering it once unregisters it no matter how many times it has been registered.
See also
  • AK::SoundEngine::UnregisterGameObj()
  • AK::SoundEngine::UnregisterAllGameObj()
  • Concept: Game Object
Parameters
in_gameObjectIDID of the game object to be registered. Valid range is [0 to 0xFFFFFFFFFFFFFFDF].
in_pszObjNameName of the game object (for monitoring purpose)
