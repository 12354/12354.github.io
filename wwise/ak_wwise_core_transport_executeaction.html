Wwise SDK 2023.1.0 - Windows
ak.wwise.core.transport.executeAction

Executes an action on the given transport object, or all transport objects if none is specified.

Arguments

Name Type Description
transport integer The transport object ID to be used with all other ak.wwise.core.transport functions.
Unsigned Integer 32-bit. Range: [0,4294967295]
action * string The action to execute. Possible values:
  • play
  • stop
  • pause
  • playStop
  • playDirectly

(Required *)
