Wwise SDK 2023.1.0 - Windows
ak.soundengine.setDefaultListeners

Sets the default active listeners for all subsequent game objects that are registered. See AK::SoundEngine::SetDefaultListeners.

Arguments

Name Type Description
listeners * array Array of listener game object IDs. Game objects must have been previously registered.
listeners [...] integer A game object ID, unsigned integer 64-bit. Range: [0,18446744073709551615]

(Required *)
