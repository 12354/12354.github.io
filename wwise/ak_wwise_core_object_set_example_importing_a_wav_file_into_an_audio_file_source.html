Wwise SDK 2023.1.0 - Windows
Importing a WAV file into an Audio File Source

Creates a Sound SFX object and its audio source. The property field Sound object is also set.

Function URI

ak.wwise.core.object.set

Arguments

{
"objects": [
{
"object": "\\Actor-Mixer Hierarchy\\Default Work Unit",
"children": [
{
"type": "Sound",
"name": "New Sound",
"children": [
{
"name": "mySource",
"type": "AudioFileSource",
"notes": "newNotes",
"import": {
"files": [
{
"audioFile": "c:\\path\\file1.wav"
}
]
}
}
]
}
]
}
]
}

Result

{}

Refer to Using the Wwise Authoring API (WAAPI) for more information about Wwise Authoring API.
