Wwise SDK 2023.1.0 - Windows
update-media-ids-in-single-file

Loads the project and updates the contents of <project-name>.mediaid, if it exists.

Examples

WwiseConsole update-media-ids-in-single-file "C:\MyProject\MyProject.wproj"

Loads the project and updates the contents of <project-name>.mediaid, if it exists.

Arguments

PROJECT

The path to the project file (.wproj).
