Wwise SDK 2023.1.0 - Windows
Pasting a list and all properties and references

Paste the property, reference and list differences from the source object to the target objects. Only the "RTPC" list will be pasted along with all properties and references. The source object only has "RTPC" and "Metadata" lists.

Function URI

ak.wwise.core.object.pasteProperties

Arguments

{
"source": "\\Actor-Mixer Hierarchy\\Default Work Unit\\Source Sound",
"targets": [
"\\Actor-Mixer Hierarchy\\Default Work Unit\\Target Sound",
"\\Actor-Mixer Hierarchy\\Default Work Unit\\Target Sound2"
],
"exclusion": [
"Metadata"
]
}

Result

{}

Refer to Using the Wwise Authoring API (WAAPI) for more information about Wwise Authoring API.
