Getting the output bus being used

Returns the inherited output bus. It uses the override parent values to determine the source of override.

Function URI

ak.wwise.core.object.get

Arguments

{
"waql": "\"{24979032-B170-43E3-A2E4-469E0193E2C3}\""
}

Options

{
"return": [
"OutputBus"
]
}

Result

{
"return": [
{
"OutputBus": {
"id": "{1514A4D8-1DA6-412A-A17E-75CA0C2149F3}",
"name": "Master Audio Bus"
}
}
]
}
