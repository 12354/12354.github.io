Wwise SDK 2023.1.0 - Windows
move-media-ids-to-work-units

Moves the project's media IDs from a single xml file <project-name>.mediaid to its work units (.wwu). This command forces a save of all the project's work units.

Examples

WwiseConsole move-media-ids-to-work-units "C:\MyProject\MyProject.wproj"

Moves a project's file IDs from a single file <project-name>.mediaid to its work units.

Arguments

PROJECT

The path to the project file (.wproj).
