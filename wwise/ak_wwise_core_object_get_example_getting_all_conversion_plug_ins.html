Wwise SDK 2023.1.0 - Windows
Getting all Conversion Plug-ins

Using a WAQL query in order to retrieve all Conversion Plug-ins.

Function URI

ak.wwise.core.object.get

Arguments

{
"waql": "from type conversionplugin"
}

Options

{
"return": [
"pluginname",
"id",
"owner.name",
"classid"
]
}
