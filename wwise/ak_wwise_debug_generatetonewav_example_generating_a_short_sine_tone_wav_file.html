Wwise SDK 2023.1.0 - Windows
Generating a short sine tone WAV file

Generates a 1 second mono WAV file that contains a sine waveform, with a very short attack and release.

Function URI

ak.wwise.debug.generateToneWAV

Arguments

{
"path": "C:\\WAV\\beep.wav",
"channelConfig": "1.0",
"waveform": "sine",
"frequency": 800,
"attackTime": 0.01,
"sustainTime": 0.98,
"releaseTime": 0.01,
"sustainLevel": 0
}

Options

{}

Result

{}
