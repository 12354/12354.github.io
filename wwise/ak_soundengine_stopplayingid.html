Wwise SDK 2023.1.0 - Windows
ak.soundengine.stopPlayingID

Stops the current content, associated to the specified playing ID, from playing. See AK::SoundEngine::StopPlayingID.

Arguments

Name Type Description
playingId * integer Playing ID to be stopped.
Unsigned Integer 32-bit. Range: [0,4294967295]
transitionDuration * integer Fade duration (ms).
Integer 32-bit. Range: [-2147483648,2147483647]
fadeCurve * integer Curve type to be used for the transition. Uses values from AkCurveInterpolation . Range: [0,9]

(Required *)
