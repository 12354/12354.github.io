Wwise SDK 2023.1.0 - Windows
ak.soundengine.stopAll

Stop playing the current content associated to the specified game object ID. If no game object is specified, all sounds are stopped. See AK::SoundEngine::StopAll.

Arguments

Name Type Description
gameObject * integer Specify a game object to stop only playback associated to the provided game object ID.
A game object ID, unsigned integer 64-bit. Range: [0,18446744073709551615]

(Required *)
