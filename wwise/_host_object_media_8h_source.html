65  * \aknote

71  * \akwarning

73  * \endakwarning
74  *
75  * \param[in] in_this Current instance of this interface.

77  * if the file is not yet available.
78  * \param[in] in_Index Media source index. If higher than or equal to the current media source count, will
79  * add a new index.
