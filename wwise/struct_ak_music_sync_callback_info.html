Detailed Description

Callback information structure corresponding to AK_MusicSyncEntry, AK_MusicSyncBeat, AK_MusicSyncBar, AK_MusicSyncExit, AK_MusicSyncGrid, AK_MusicSyncPoint and AK_MusicSyncUserCue. If you need the Tempo, you can compute it using the fBeatDuration Tempo (beats per minute) = 60/fBeatDuration
