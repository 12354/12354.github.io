Stopping a current content playing

Stops the content, specified by "playingId", by using "transitionDuration" with a Linear Curve.

Function URI

ak.soundengine.stopPlayingID

Arguments

{
"playingId": 5566998,
"transitionDuration": 2000,
"fadeCurve": 4
}

Result

{}
