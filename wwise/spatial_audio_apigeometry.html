Wwise SDK 2023.1.0 - Windows
Geometry

The Geometry API allows the game to send a triangle mesh to Wwise Spatial Audio, for two purposes:

  • Simulating early reflections using Reflect: Using the Geometry API for Simulating Early Reflections.
  • Simulating edge diffraction of sound propagating from emitters to listener: Using the Geometry API for Simulating Diffraction and Transmission.
