Wwise SDK 2023.1.0 - Windows
Gets the RTPC ids from a sound's "RTPC" list

Uses the list accessor @ to reference the "RTPC" list. Note that RTPCs are unnamed.

Function URI

ak.wwise.core.object.get

Arguments

{
"waql": "\"{24979032-B170-43E3-A2E4-469E0193E2C3}\""
}

Options

{
"return": [
"@RTPC"
]
}

Result

{
"return": [
{
"@RTPC": [
{
"id": "{67A2C1B5-4FBF-4938-A86C-4AEE7E84822A}"
},
{
"id": "{30D99E75-A579-4163-8FEC-012985D0B451}"
}
]
}
]
}
