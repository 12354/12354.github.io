AkReal32 AK::AkMetering::fMeanPowerK

Mean k-weighted power value in this frame, used to compute loudness (as defined by ITU-R BS.1770). Total linear k-weighted power of all channels. 0 if AK_EnableBusMeter_KPower is not set (see IAkMixerPluginContext::SetMeteringFlags() or AK::SoundEngine::RegisterBusMeteringCallback()).
