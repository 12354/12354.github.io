virtual AkUInt32 AK::StreamMgr::IAkLowLevelIOHook::GetDeviceData ( )
pure virtual

Returns custom profiling data for the streaming device above this low-level hook. As opposed to GetDeviceDesc(), this is called at every monitoring frame. You may implement this function in order to display any value you find useful in the "Streaming Devices" tab of the Wwise profiler ("Custom Param" column).

Remarks
For profiling purposes only. The Release configuration of the Stream Manager never calls it.
Returns
A 32-bit unsigned value to display in the Wwise profiler.
