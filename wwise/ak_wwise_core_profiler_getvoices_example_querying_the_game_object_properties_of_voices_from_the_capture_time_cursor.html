Wwise SDK 2023.1.0 - Windows
Querying the game object properties of voices from the Capture Time Cursor

Specifies the Capture Time Cursor as a time to query the voices at the latest time of the capture session when the call is processed.

Function URI

ak.wwise.core.profiler.getVoices

Arguments

{
"time": "capture"
}

Options

{
"return": [
"gameObjectID",
"gameObjectName"
]
}

Result

{
"return": [
{
"gameObjectID": 18446744073709551614,
"gameObjectName": "Transport/Soundcaster"
}
]
}

Refer to Using the Wwise Authoring API (WAAPI) for more information about Wwise Authoring API.
