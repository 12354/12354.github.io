virtual AKRESULT AK::IAkOutOfPlaceEffectPlugin::TimeSkip ( AkUInt32 &  io_uFrames)
pure virtual

Skips execution of some frames, when the voice is virtual playing from elapsed time. This can be used to simulate processing that would have taken place (e.g. update internal state). Return AK_DataReady or AK_NoMoreData, depending if there would be audio output or not at that point.

Parameters
io_uFramesNumber of frames the audio processing should advance. The output value should be the number of frames that would be consumed to output the number of frames this parameter has at the input of the function.
