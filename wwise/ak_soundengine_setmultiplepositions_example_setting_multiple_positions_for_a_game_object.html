Wwise SDK 2023.1.0 - Windows
Setting multiple positions for a game object

Sets "gameObject" to multiple positions, as defined in "positions", to simulate multiple sources for one playing sound.

Function URI

ak.soundengine.setMultiplePositions

Arguments

{
"gameObject": 1122334,
"positions": [
{
"position": {
"orientationFront": {
"x": 5,
"y": 20,
"z": 10
},
"orientationTop": {
"x": 70,
"y": 150,
"z": 6
},
"position": {
"x": 70,
"y": 150,
"z": 10
}
}
},
{
"position": {
"orientationFront": {
"x": -50,
"y": 80,
"z": 15
},
"orientationTop": {
"x": 2,
"y": 40,
"z": 6
},
"position": {
"x": 2,
"y": -20,
"z": 400
}
}
}
],
"multiPositionType": 1
}

Result

{}
