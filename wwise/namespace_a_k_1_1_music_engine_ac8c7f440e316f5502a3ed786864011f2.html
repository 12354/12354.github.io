AKSOUNDENGINE_API AKRESULT AK::MusicEngine::Init ( AkMusicSettings *  in_pSettings)

Initialize the music engine.

Warning
This function must be called after the base sound engine has been properly initialized. There should be no AK API call between AK::SoundEngine::Init() and this call. Any call done in between is potentially unsafe.
Returns
AK_Success if the Init was successful, AK_Fail otherwise.
