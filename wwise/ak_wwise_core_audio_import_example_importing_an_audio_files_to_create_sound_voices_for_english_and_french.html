Wwise SDK 2023.1.0 - Windows
Importing an audio files to create Sound Voices for English and French

Imports files using an absolute objectPath and specifying the language for each entry.

Function URI

ak.wwise.core.audio.import

Arguments

{
"importOperation": "useExisting",
"imports": [
{
"audioFile": "D:\\sources\\hello_en.wav",
"objectPath": "\\Actor-Mixer Hierarchy\\Default Work Unit\\<Sound Voice>hello\\<AudioFileSource>hello_en",
"importLanguage": "EN_US"
},
{
"audioFile": "D:\\sources\\hello_fr.wav",
"objectPath": "\\Actor-Mixer Hierarchy\\Default Work Unit\\<Sound Voice>hello\\<AudioFileSource>hello_fr",
"importLanguage": "FR_CA"
}
]
}

Options

{}

Result

{
"objects": [
{
"id": "{D26E5952-458B-4143-9322-4EA5C99B5A75}",
"name": "hello"
},
{
"id": "{01467DBD-081B-4362-8569-B7E6A27E9965}",
"name": "hello_en"
},
{
"id": "{1FBC2142-6388-494A-A849-D272190B095D}",
"name": "hello_fr"
}
]
}
