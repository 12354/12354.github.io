AkUInt16 AkCommSettings::Ports::uDiscoveryBroadcast

This is where the authoring application broadcasts "Game Discovery" requests to discover games running on the network. Default value: 24024.

Warning
Unlike the other port in this structure, this port cannot be dynamic: setting it to 0 will disable discovery. Refer to One Fixed Port: Discovery Broadcast for more details.
