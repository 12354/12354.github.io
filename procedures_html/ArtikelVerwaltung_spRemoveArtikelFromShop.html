ArtikelVerwaltung.spRemoveArtikelFromShop

Description: Die Procedure löscht die übergebenen Artikel aus allen Shops.

Long Description: Keine ausführliche Beschreibung hinterlegt.

Created At: 2024-01-12 03:32:03

Updated At: 2024-01-12 03:32:03

Code:

CREATE PROCEDURE ArtikelVerwaltung.spRemoveArtikelFromShop 
	@artikelKeys ArtikelVerwaltung.TYPE_spArtikelRemoveFromShop READONLY
AS
DELETE dbo.tArtikelShop
FROM dbo.tArtikelShop 
JOIN @artikelKeys ON [@artikelKeys].kArtikel = tArtikelShop.kArtikel
