Column Name Type Nullable Description Created At Updated At
cProgressText NVARCHAR(512) Yes Meldung des Abgleichs. Diese Meldung wird in der UI angezeigt.

kMandant INT No Id des Mandanten bzw. -1 wenn Mandantenweit.

nStatus INT No Gibt den aktuellen Status an: 0=Beendet, wird aber beim nächsten Start automatisch ausgeführt, 1=Wird gestartet, 2=Wird ausgeführt, 3=Erfolgreich ausgeführt, 4=Mit Fehlern ausgeführt, 5=Wird gestoppt, 6=Gesperrt, 7=Deaktiviert.
