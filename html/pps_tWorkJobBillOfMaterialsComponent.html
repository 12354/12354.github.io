Column Name Type Nullable Description Created At Updated At
bRowversion TIMESTAMP No Keine Beschreibung hinterlegt.
fActualQuantity DECIMAL(25, 13) Yes Die Istmenge oder null, wenn für den zugrunde liegenden Artikel eine Bestandsführung konfiguriert ist, die bedingt, dass die Istmenge anhand der zu belastenden Bestände summiert werden muss.
fTargetQuantity DECIMAL(25, 13) Yes Die Sollmenge oder null, wenn die Sollmenge der Menge der zugrunde liegenden Stücklistenkomponente entspricht.
kArtikel INT No Der Schlüssel des zugrunde liegenden Artikels.
kMasseinheit INT Yes Der Schlüssel der Maßeinheit, in der fActualQuantity und fTargetQuantity geführt sind, oder null, wenn sie in der Verkaufseinheit des zugrunde liegenden Artikels geführt sind.
kOriginBillOfMaterialComponent INT Yes Der Schlüssel der zugrunde liegenden Stücklistenkomponente oder null, wenn die Artikelposition anderweitig entstanden ist.
kWorkJobBillOfMaterialsComponent INT No Der Schlüssel, der diese Artikelposition eindeutig identifiziert.
kWorkJobOperation INT No Der Schlüssel des Arbeitsgangs, in dem diese Artikelposition enthalten ist.
