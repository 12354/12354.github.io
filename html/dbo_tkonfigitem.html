dbo.tkonfigitem Details

Column Name Type Nullable Description Created At Updated At
bEmpfohlen TINYINT Yes Gibt an, ob das Konfigurationsitem standardmäßig hervorgehoben wird (True/False).
bIgnoreMultiplier TINYINT No Gibt an, ob der Artikelmultiplignoriert werden soll. Wenn ja, wird die Menge nicht davon beinflusst.
bName TINYINT Yes Gibt an, ob der Artikelname und die zugehörige Beschreibung übernommen werden sollen, wenn die Artikel ID gesetzt wurde (True/False).
bPreis TINYINT Yes Gibt an, ob sich der Artikelpreis von seinem referenziertem Artikel ableitet.
bRabatt TINYINT Yes Gibt an, ob der Rabatt auf den Artikelpreis angezeigt wird.
bRowversion TIMESTAMP No Keine Beschreibung hinterlegt.
bSelektiert TINYINT Yes Gibt an, ob das zugehörige Konfigurationsiten standardmäßig ausgewält ist.
bZuschlag TINYINT Yes Optionales Feld: Gibt an, ob der Zuschlag auf den Produktpreis angezeigt wird.
fInitial DECIMAL(25, 13) No Enthält die optionale Menge.
fMax DECIMAL(25, 13) No Enthält das erlaubte Maximum (0 für kein Limit).
fMin DECIMAL(25, 13) No Gibt die benötigte Mindesmenge an, um das Konfigurationsitem hinzuzufügen.
fStandardpreis DECIMAL(25, 13) No Enthält den Standardpreis
kArtikel INT Yes Verweis auf den Primärschlüssel der Tabelle dbo.tArtikel
kKonfiggruppe INT Yes Verweis auf den Primärschlüssel der Tabelle dbo.tKonfiggruppe
kKonfigitem INT No Primärschlüssel der Tabelle
nPosTyp INT Yes Enthält den Typ des Items (0 für ein Produkt, 1 für Spezial).
nSort INT Yes Enthält die Sortierungsnummer.
